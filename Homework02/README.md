# 《基于SVHN数据集比较CNN,ResNet.SE-ResNet性能》实验报告

## 1. 实验目的

本实验以 SVHN 门牌号数字识别十分类任务为对象，对比三种网络在相同数据集上的分类表现：

- `CNN` 基线网络
- `ResNet` 轻量残差网络
- `SE-ResNet` 引入通道注意力机制的轻量残差网络

实验目标是分析三种模型在整体精度、测试损失、泛化表现和分类别识别能力上的差异，并据此总结不同结构设计对识别结果的影响。


## 2. 模型说明
### 2.1 CNN
  `CNN` 采用普通卷积堆叠结构。网络由 `3` 个卷积模块和 `2` 层全连接组成，`base_channels=32`。其中三个卷积模块的通道数依次为 `3→32`、`32→64`、`64→128`，每个模块内部均为 `3×3` 卷积、`BatchNorm`、`ReLU` 和 `2×2 MaxPool`。输入图像大小为 `32×32`，经过三次池化后，特征图尺寸变为 `4×4`，再展平后送入分类器。分类头结构为 `Linear(128×4×4, 256) -> ReLU -> Dropout(0.3) -> Linear(256, 10)`。

### 2.2 Resnet
 `ResNet-111` 是一个针对小尺寸图像调整后的轻量残差网络。采用的 stage 深度为 `[1,1,1]`，保留 `3` 个残差阶段，每个阶段包含 `1` 个 `BasicBlock`，同时 `base_channels=32`。其整体结构如下：
- Stem：`3 -> 32`，`3×3` 卷积，步长 `1`
- Layer1：`32 -> 32`，包含 `1` 个残差块，步长 `1`
- Layer2：`32 -> 64`，包含 `1` 个残差块，首块步长 `2`
- Layer3：`64 -> 128`，包含 `1` 个残差块，首块步长 `2`

每个 `BasicBlock` 的内部结构为：

`Conv(3×3) -> BatchNorm -> ReLU -> Conv(3×3) -> BatchNorm -> 残差相加 -> ReLU`

当通道数变化或步长不为 `1` 时，残差支路会通过一个 `1×1` 卷积加 `BatchNorm` 进行下采样和通道对齐，即：

`Conv(1×1, stride=2) -> BatchNorm`

主干特征提取完成后，网络使用：

- `AdaptiveAvgPool2d(1,1)`
- `Linear(128 -> 10)`

作为最终分类头。

### 2.3 SE-ResNet
`SE-ResNet` 的整体骨架与 `ResNet` 完全一致，同样采用 `[1,1,1]` 的三阶段残差结构，二者的主要区别在于：`SE-ResNet-111` 在每个残差块中加入了 `SEBlock` 通道注意力模块。`SEBlock` 的实现流程为：
1. 对输入特征图做全局平均池化，将空间信息压缩为通道描述子
2. 经过两层全连接实现通道权重学习
3. 使用 `Sigmoid` 得到每个通道的权重
4. 将通道权重重新作用到原特征图上

### 3.2 对比指标

从以下角度展开分析：

- 最佳验证准确率：反映训练过程中模型能达到的最好验证性能。
- 测试准确率：反映模型在测试集上的最终分类能力。
- 测试损失：反映模型输出概率分布与真实标签之间的偏差程度。
- 泛化差距：以最终训练准确率减去最终验证准确率，用于观察过拟合倾向。
- 分类别准确率：用于分析不同模型在具体数字类别上的优势与短板。

## 4. 总体实验结果

| 模型  | 最佳验证准确率 | 测试准确率 | 测试损失 | 泛化差距 |
| --- | --- | --- | ---: | ---: | ---: | ---: | ---: | ---: |
| ResNet | 93.02% | 93.83% | 0.2209 | 4.26% |
| SE-ResNet | 93.23% | 93.79% | 0.2174 | 3.13% |
| CNN | 92.86% | 92.29% | 0.2604 | 5.79% | 

从总体结果看，`ResNet` 取得了最高测试准确率 `93.83%`，说明其在当前实验条件下具有最强的整体识别能力。进一步比较可以发现：

1. `ResNet` 相对 `CNN` 提升了 `1.54%`，说明引入残差结构后，模型对 SVHN 任务的表征能力有了明显增强。
2. `ResNet` 相对 `SE-ResNet` 仍领先 `0.04%`。这一差距很小，说明在相同 `base_channels=32` 条件下，两种残差模型的整体性能已经非常接近。
3. `SE-ResNet` 的测试损失比 `ResNet-111` 低 `0.0035`，泛化差距也更小 `1.13%`，说明它虽然没有拿到最高准确率，但输出分布更稳定，过拟合倾向略弱。


## 5. 分类别结果分析

| 数字类别 | CNN | ResNet | SE-ResNet | 最佳模型 |
| --- | ---: | ---: | ---: | --- |
| 0 | 92.09% | 94.04% | 93.06% | ResNet-111 |
| 1 | 95.49% | 95.86% | 96.25% | SE-ResNet-111 |
| 2 | 94.91% | 95.90% | 95.35% | ResNet-111 |
| 3 | 89.17% | 90.28% | 92.30% | SE-ResNet-111 |
| 4 | 94.41% | 96.39% | 95.72% | ResNet-111 |
| 5 | 91.90% | 93.29% | 91.69% | ResNet-111 |
| 6 | 89.83% | 93.42% | 92.21% | ResNet-111 |
| 7 | 92.27% | 93.02% | 93.16% | SE-ResNet-111 |
| 8 | 85.24% | 87.53% | 90.78% | SE-ResNet-111 |
| 9 | 88.78% | 92.98% | 91.29% | ResNet-111 |

分类别结果能够更细致地反映模型差异：

1. `ResNet` 在 `6/10` 个类别上取得最好结果，具体为 `0、2、4、5、6、9` 六个类别，说明其整体均衡性更强。
2. `SE-ResNet` 在 `4/10` 个类别上领先，即 `1、3、7、8` 四个类别，说明 SE 注意力机制对部分困难类别更有帮助。
3. 在数字 `8` 上，`SE-ResNet` 相比 `ResNet` 高 `3.25%`。这类样本通常笔画更密集、局部模式更复杂，SE 模块通过重标定通道响应，可能更有利于提取关键区域特征。
4. `CNN` 在本次对比中没有获得单类第一，表明单纯依赖浅层卷积堆叠时，模型对复杂类别细节的区分能力仍然有限。

综合来看，`ResNet` 更偏向于提升整体精度与类别均衡性，而 `SE-ResNet` 更偏向于改善部分困难类别的鲁棒性与稳定性。这也是二者总体准确率接近、但单类表现分布不同的主要原因。

## 7. 结论

1. 在本次 `base_channels=32` 的公平比较条件下，`ResNet-111` 取得最高测试准确率，是三种模型中综合表现最好的方案。
2. `SE-ResNet` 与 `ResNet` 的总体准确率差距非常小，但测试损失更低、泛化更稳定，说明其优势更多体现在模型鲁棒性与困难类别识别上。
3. `CNN` 训练最快、结构最简单，适合作为基线模型；但从最终结果看，其性能明显落后于两种残差结构，说明在 SVHN 任务中，引入残差连接比单纯使用普通卷积堆叠更有效。
